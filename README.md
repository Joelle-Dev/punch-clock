# 打卡记录 Punch

一个极简的多类型打卡 Web 应用（支持：如厕 / 饭否 / 健身 / 其他），数据保存在本机浏览器，可导出为 CSV。

---

## 功能简介

- **多类型打卡**：顶部类型页签（如厕 / 饭否 / 健身 / 其他），各类型独立统计  
- **打卡历史与统计**：记录精确到秒，支持今日次数、连续天数等基础统计  
- **筛选与管理**：按时间范围与类型筛选，支持导出 CSV、清空全部记录  
- **本地存储**：所有数据仅保存在当前浏览器的 `localStorage`

---

## 使用方式

- **直接打开**
  1. 打开项目文件夹 `punch`
  2. 用浏览器打开 `index.html` 即可使用

- **本地开发（Vite）**
  ```bash
  npm install
  npm run dev   # 默认 http://localhost:3000
  ```
