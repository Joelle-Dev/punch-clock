# 打卡记录 Punch

一个极简的多类型打卡 Web 应用（支持：拉粑粑 / 饭否 / 健身 / 其他），数据保存在本机浏览器，可导出为 CSV。

---

## 功能特性

- **多类型打卡**
  - 顶部类型页签：拉粑粑 / 饭否 / 健身 / 其他
  - 不同类型独立统计，切换类型后再打卡

- **打卡记录**
  - 精确到「年月日 时:分:秒」
  - 「打卡概览与历史」中按类型块、按日期分组展示
  - 每个时间点都可以单独删除，带二次确认弹窗（自定义样式，而不是系统 `confirm`）

- **统计信息**
  - 今日打卡次数
  - 连续打卡天数（按自然日连续计算）

- **筛选视图**
  - 时间筛选：全部 / 今天 / 本周 / 本月
  - 类型筛选：全部类型 / 拉粑粑 / 饭否 / 健身 / 其他

- **数据导出与清空**
  - 导出当前所有记录为 CSV（含类型、日期、时间、时间戳）
  - 一键清空全部记录（带二次确认）

- **本地存储**
  - 所有数据存于浏览器 `localStorage`
  - 只在当前设备、当前浏览器中有效，**不会自动跨设备同步**

---

## 使用说明

### 本地打开（零配置）

1. 打开项目文件夹 `punch`
2. 用浏览器直接打开 `index.html` 即可使用  
   （双击 `index.html` 或拖进浏览器窗口）

### 本地开发（Vite）

# 安装依赖
npm install

# 启动开发服务器（默认 http://localhost:3000）
npm run dev
